# Live Site Snapshots

This folder stores dated backups of the public production site as fetched from `https://opticable.ca`.

- Each snapshot is a mirror of the live public pages and same-origin assets at the time it was captured.
- Snapshots are archival only and are not used by `sitegen.py` or the live build.
- External third-party URLs are recorded in each snapshot manifest but are not mirrored into the repository.

The current production backup created before further site changes is stored in the dated subfolder next to this file.
