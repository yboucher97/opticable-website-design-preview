# Spring Promo Social Source

Source files for the Concept 01 spring promo social exports.

- `spring-promo-social.html`
  Reusable master artwork with format switching through the `format` query parameter.

- `export-social-assets.ps1`
  Exports platform-ready PNG files into:
  `Images/concepts/concept-01/generated-candidates/spring-promo-2026-04/`

Supported formats:

- `square`
  2160 x 2160 for Instagram and Facebook feed

- `portrait`
  2160 x 2700 for taller Instagram and Facebook feed placement

- `story`
  2160 x 3840 for Stories or reel cover use

- `linkedin`
  2400 x 1254 for LinkedIn and wide social placements
