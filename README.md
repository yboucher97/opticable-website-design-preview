# Opticable Website

Static bilingual production website source for Opticable.

The site is generated by `sitegen.py` and deployed with Cloudflare using the generated `dist/` directory plus `worker.js`.

Concept previews and archived generated snapshots live in `yboucher97/opticable-website-concepts` so this repository stays focused on production.

## Project structure

- `sitegen.py`: canonical source for page content, HTML rendering, CSS/JS output, and image processing.
- `assets/`: production runtime brand assets copied or transformed into `dist/assets/`.
- `Images/production/`: production-only image sources used by the live generator.
- `Images/`: image workflow documentation. See `Images/README.md`.
- `dist/`: generated deployment output served by Wrangler.
- `worker.js`: Cloudflare Worker that adds redirects, cache headers, security headers, and custom 404 handling.
- `wrangler.jsonc`: deployment configuration.

## Build locally

1. `python -m venv .venv`
2. `.venv\\Scripts\\activate`
3. `pip install -r requirements.txt`
4. `python sitegen.py`

## Deploy

- `wrangler deploy`
