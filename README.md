# Opticable Website

Static bilingual website source for Opticable.

The site is generated by `sitegen.py` and deployed with Cloudflare using the generated `dist/` directory plus `worker.js`.

## Project structure

- `sitegen.py`: canonical source for page content, HTML rendering, CSS/JS output, and image processing.
- `assets/`: runtime source assets copied or transformed into `dist/assets/`.
- `Images/`: raw/reference image library used as source material for generated website visuals.
- `dist/`: generated deployment output served by Wrangler.
- `worker.js`: Cloudflare Worker that adds redirects, cache headers, security headers, and custom 404 handling.
- `wrangler.jsonc`: deployment configuration.

## Build locally

1. `python -m venv .venv`
2. `.venv\\Scripts\\activate`
3. `pip install -r requirements.txt`
4. `python sitegen.py`

## Deploy

- `wrangler deploy`
