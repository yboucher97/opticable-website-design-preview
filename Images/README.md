# Image Folder Guide

This repository uses a strict split between production, concepts, and archive.

## Production

- `production/`
  Live-site image sources only.
  If `sitegen.py` uses an image, it should come from here or from `assets/`.

## Concepts

- `concepts/concept-01/`
- `concepts/concept-02/`
- `concepts/concept-03/`

Each concept folder contains:

- `generated-candidates/`
  New images generated by Codex for that concept.

- `manual-upload/`
  Files manually added for that concept.

- `selected/`
  Images currently chosen for that concept.

## Archive

- `archive/`
  Old references, backups, unused variants, and research material.
  Nothing here should be treated as live.

## Promotion rule

Do not point `sitegen.py` at `concepts/` or `archive/`.

When a concept image becomes a winner:

1. Move or copy it into `production/`.
2. Update `sitegen.py`.
3. Rebuild the site.

## Deprecated

- `future-use/`
  Deprecated shared intake area from the earlier workflow.
  Use per-concept folders instead.
